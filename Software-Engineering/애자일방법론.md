    ex) 이거 바꿔주세용

```
"소프트웨어의 모든 것은 변한다."

변화는 반드시 일어나기 때문에, 문제가 되는 것은 변화가 아니다. 그보다는 변화를 극복하지 못하는 우리의 무능력이 문제다.
```

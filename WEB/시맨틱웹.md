# 시맨틱웹

## 의미
시맨틱(semantic) : 의미론적<br/>
즉, 시맨틱 HTML을 직역하면 의미론적인 HTML이다. 문서의 의미에 맞게, 어플리케이션의 의미에 맞게 HTML을 작성하는 것을 뜻한다.

## 태그의 종류

1. `<main>` 태그

2. `<section>` 태그

3. `<article>` 태그
